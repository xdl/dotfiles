# Dotfiles

Configuration files/scripts for various tools and shells

## {Li,U}nix

Initial setup:

```
chmod u+x setup.sh
./setup.sh
```

For vim:

```
chmod u+x vim/setup.sh
vim/setup.sh
```

For fonts:

    # Chinese
    sudo apt install xfonts-wqy
    
    # DeJaVu
    sudo apt-get install fonts-dejavu
